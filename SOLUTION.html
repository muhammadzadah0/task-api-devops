Cara deploy:

kubectl apply -f k8s/namespace.yaml
kubectl apply -f k8s/

Helm Chart

helm/task-api/

  values-bluegreen.yaml      # Blue-green mode
  templates/
    _helpers.tpl             # Helper functions
    deployment.yaml          # Main deployment + service
    bluegreen.yaml           # Preview deployment + dual services

Blue-Green Deployment Strategy

  1. Blue (active) — versi lama melayani traffic
  2. Green (preview) — versi baru di-deploy, tidak terima traffic
  3. Test & validasi green melalui preview service
  4. Switch — alihkan service selector dari blue ke green
  5. Rollback — kembalikan selector ke blue (masih running)

Coba blue-green:

helm install task-api helm/task-api -f helm/task-api/values-bluegreen.yaml

Switch traffic:

kubectl patch service task-api-blue -p '{"spec":{"selector":{"app":"task-api-preview"}}}'

Rollback:

kubectl patch service task-api-blue -p '{"spec":{"selector":{"app":"task-api"}}}'
